## 🔧 TECHNICAL DETAILS

### Technology Stack
- **Backend**: Laravel 11, PHP 8.2+
- **Frontend**: React 18+, TypeScript
- **Database**: MySQL/MariaDB
- **State Management**: Inertia.js
- **Styling**: Tailwind CSS
- **UI Library**: Custom Tailwind components

### Code Patterns Used
```php
// Controller Pattern
public function index(Request $request) {
    $affiliate = $request->user()->affiliate;
    $data = Model::where('affiliate_id', $affiliate->id)
        ->paginate(15);
    return Inertia::render('Page', ['data' => $data]);
}

// Component Pattern
export default function Component({ data }: Props) {
    return (
        <AppLayout>
            {data.map(item => <TableRow key={item.id} />)}
        </AppLayout>
    );
}
```

### Database Models Connected
- User ↔ Affiliate (1:1)
- Affiliate ↔ Commission (1:many)
- Affiliate ↔ Order (1:many)
